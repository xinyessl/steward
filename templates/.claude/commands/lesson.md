description: 把刚解决的一个坑沉淀进 Steward 全局经验库（$STEWARD_LESSONS，跨所有项目共享）
---
把「$ARGUMENTS」（没写就用我们**刚在本会话解决的那个问题**）记成一条经验，追加到 **Steward 全局经验库**。

步骤：
1. 解析路径：`echo "$STEWARD_LESSONS"`（即 `~/.steward/lessons.md`），`cat "$STEWARD_LESSONS"` 读现有内容、看最大编号。
2. 在「📚 教训明细」末尾追加一条（编号顺延 L-NNN）：
   - **现象**（一句话）/ **范围·模块**（哪个项目/功能/文件）/ **根因** / **解法** / **防复发**（下次怎么避免）/ **关联**（spec id / 提交 / 无）。
3. 若有可复用的检查点，往「✅ 自检清单」加一条。
4. 用 Edit/Write 写回 `$STEWARD_LESSONS` 这个**绝对路径文件**——⚠️ 别在当前项目里另建 lessons.md（经验库是全局共享的、跨项目）。
5. 回我一句：`已记录 L-NNN：<现象>`。

只记**真实发生、确认有效**的解法；模糊/没验证的不要记，宁缺毋滥。
